`pimp`
======

`pimp` is a simple command lint utility to calculate the "Performance IMProvement"
of a `before` and `after` figure

Example
-------

    $ pimp 436.8 90.5
    Before:  436.80
    After:   90.50
    Delta:   346.30
    Change:  79.28%

Installation
------------

    go get github.com/maurice/pimp
